    Nikola

    The Nikola Handbook

    nikola init site

    Inizializza un nuovo sito creando la directory site

    nikola new_post

    Crea un nuovo post

    nikola new_post -p

    Crea una nuova pagina

    nikola new_post -f ipynb

    Crea un nuovo post in formato Jupyter notebook

    nikola build

    Costruisce il sito

    nikola serve -b

    Avvia il web server e apre un browser

    nikola github_deploy

    Distribuisce il sito su github

    .. TEASER_END

    Conda

    Conda Cheat Sheet

    conda info

    Informazioni sull'installazione

    conda install / update package

    Installa / aggiorna il pacchetto package

    conda update --all

    Aggiorna tutti i pacchetti

    conda create -n name packages

    Crea un ambiente name e installa i pacchetti packages

    conda activate name / deactivate

    Attiva / disattiva un ambiente

    conda env list

    Mostra tutti gli ambienti disponibili

    conda list --revisions

    Mostra la cronologia degli aggiornamenti

    conda config --add channels channel

    Aggiunge il canale channel all'elenco dei canali

    Git

    Git Cheat Sheet

    git init

    Inizializza un repository nella directory attiva

    git clone url

    Clona un repository da un URL remoto

    git add file/directory

    Aggiunge un file / una directory ai cambiamenti da validare

    git commit -m message

    Valida i cambiamenti effettuati con il messaggio message

    git status

    Mostra la suddivisione dei file tra validati, da validare e non inclusi

    git remote add name url

    Aggiunge un'origine remota a un repository locale, dall'URL url e con l'alias name

    git fetch remote (branch)

    Scarica i dati da un repository remoto, opzionalmente da un branch specificato

    git pull remote

    Scarica i dati dal branch attivo del repository remoto unendoli immediatamente alla directory locale

    git push remote branch

    Invia i dati del branch locale al repository remoto

    IPython Documentation

    python -m ipykernel --user --name environment --display-name name

    Installa un kernel di Python nell'ambiente environment con il nome name

    reStructuredText

    Quick reStructuredText

    *abc*

    Testo in corsivo

    **abc**

    Testo in grassetto

    `abc`

    Testo interpretato (vari utilizzi)

    ``abc``

    Testo letterale (in linea)

    abc_

    Riferimento (hyperlink)

    `abc def`_

    Riferimento se presenti spazi o segni di punteggiatura

    `abc def <http://xyz.com>`_

    Riferimento con URI incorporato

    _abc

    Target del riferimento (in linea)

    .. _abc:

    Target del riferimento interno

    .. _abc: http://xyz.com

    Target del riferimento con URI

    [#]_

    Note a piè di pagina (# per auto-numerazione)

    .. [#]

    Target della nota a piè di pagina

    .. contents::
